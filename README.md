# E-Commerce Sales Analytics using SQL

## Project Overview

This project uses SQL to analyze e-commerce sales data and generate business insights related to revenue, profit, customer behavior, and product performance.

The objective is to demonstrate how SQL can be used to answer real-world business questions and support data-driven decision-making.

## Key Insights

* A small percentage of customers generated a large share of total revenue.
* Certain categories consistently delivered higher sales performance.
* Monthly revenue trends revealed growth opportunities and seasonality.
* Product-level analysis identified key revenue-driving items.
